# Exports

`tiptap-react-ui` exposes a small public API. This page lists the supported import surface for the package.

## Runtime Exports

```tsx
import {
  RichTextEditor,
  NotionEditor,
  useTiptapEditor,
  RenderJSON,
  RenderHTML,
  RICH_TEXT_EDITOR_THEMES,
} from "tiptap-react-ui";
```

| Export | Kind | Purpose |
| --- | --- | --- |
| `RichTextEditor` | Component | Renders the full editor UI, toolbar, bubble menus, preview tabs, save action, theme controls, extension state controls, and word count footer. |
| `NotionEditor` | Component | Renders a Notion-like editor with slash commands, block controls, contextual menus, and extension state controls. |
| `useTiptapEditor` | Hook | Creates the preconfigured Tiptap editor instance, returns `isEditorLoading`, and returns helpers for reading content. |
| `RenderJSON` | Component | Renders saved Tiptap JSON content in a read-only editor surface. Recommended for editor-generated content. |
| `RenderHTML` | Component | Renders saved HTML content, with optional sanitization. |
| `RICH_TEXT_EDITOR_THEMES` | Constant | List of built-in RichTextEditor theme names. |

## Type Exports

```ts
import type {
  NotionEditorProps,
  OnSaveArgs,
  RichTextEditorMode,
  RichTextEditorProps,
  RichTextEditorTheme,
} from "tiptap-react-ui";
```

| Export | Purpose |
| --- | --- |
| `NotionEditorProps` | Props for `NotionEditor`. |
| `OnSaveArgs` | Payload returned by `RichTextEditor` when `onSave` is triggered. |
| `RichTextEditorMode` | `"light"`, `"dark"`, or `"system"`. |
| `RichTextEditorProps` | Props for `RichTextEditor`. |
| `RichTextEditorTheme` | Built-in RichTextEditor theme names. |

## Stylesheet Exports

Import one of the package stylesheet entry points once in your root layout, app entry, or app-level stylesheet.

```tsx
import "tiptap-react-ui/style.css";
```

```tsx
import "tiptap-react-ui/dist/index.css";
```

The package ships compiled styles. Consumer apps do not need Tailwind CSS or shadcn/ui configured for the editor UI to work.

## Recommended Setup

```tsx
"use client";

import "tiptap-react-ui/style.css";
import { RichTextEditor, useTiptapEditor } from "tiptap-react-ui";

export function EditorExample() {
  const { editor, isEditorLoading } = useTiptapEditor();

  if (isEditorLoading) return null;

  return <RichTextEditor editor={editor} />;
}
```

## Import Notes

Only the root package exports are considered public API. Internal files under `dist` or source-like paths should not be imported directly because they can change without a public API change.
