# Types & Interfaces

Reference for the public TypeScript types used by `tiptap-react-ui`.

## TiptapEditorProps

Props accepted by `useTiptapEditor`.

```ts
type TiptapEditorProps = {
  handleImageUpload?: {
    onUpload?: (
      file: File,
      onProgress?: (event: { progress: number }) => void,
      abortSignal?: AbortSignal,
    ) => Promise<string>;
    maxLimit?: number;
    limit?: number;
  };
  tableOfContents?: TableOfContentsOptions;
  immediatelyRender?: boolean;
  className?: string;
  placeholder?: string;
} & TiptapOptions;
```

| Property | Type | Default | Description |
| --- | --- | --- | --- |
| `handleImageUpload` | `object` | fallback uploader | Configures the image upload node. |
| `tableOfContents` | `TableOfContentsOptions` | - | Registers and configures the Tiptap table of contents extension. |
| `immediatelyRender` | `boolean` | `false` | Controls immediate rendering. Defaults to `false` for SSR safety. |
| `className` | `string` | - | Classes applied to the editable ProseMirror element. |
| `placeholder` | `string` | `"Write something..."` | Placeholder shown when the editor is empty. |

`useTiptapEditor` also accepts selected Tiptap `UseEditorOptions` such as `content`, `editable`, `autofocus`, lifecycle hooks, parsing options, and rendering options.

## TiptapEditorReturnTypes

```ts
type TiptapEditorReturnTypes = {
  editor: Editor;
  isEditorLoading: boolean;
  getEditorContent: () => {
    json: JSONContent | null;
    html: string | null;
  };
};
```

| Property | Type | Description |
| --- | --- | --- |
| `editor` | `Editor` | Tiptap editor instance. It can be unavailable during initial mount. |
| `isEditorLoading` | `boolean` | `true` while the editor instance is initializing. |
| `getEditorContent` | `Function` | Returns current content in JSON and HTML formats. |

## RichTextEditorProps

```ts
type RichTextEditorProps = {
  editor: Editor;
  onSave?: (val: OnSaveArgs) => void;
  enablePreview?: boolean;
  enableTableOfContents?: boolean;
  tableOfContentsPosition?: "left" | "right";
  enableWordCount?: boolean;
  immediatelyRenderPreview?: boolean;
  extensionState?: EditorExtensionStateMap;
  enableModeToggle?: boolean;
  mode?: RichTextEditorMode;
  theme?: RichTextEditorTheme;
  wrapperClassName?: string;
  editorClassName?: string;
};
```

| Property | Default | Description |
| --- | --- | --- |
| `editor` | Required | Editor instance from `useTiptapEditor`. |
| `onSave` | - | Adds a save action and receives `{ json, html }`. |
| `enablePreview` | `false` | Shows Editor and Preview tabs. |
| `enableTableOfContents` | `false` | Shows a generated TOC in JSON preview. |
| `tableOfContentsPosition` | `"right"` | Controls TOC placement. |
| `enableWordCount` | `true` | Shows word and character count footer. |
| `immediatelyRenderPreview` | `false` | Controls preview rendering timing. |
| `extensionState` | all visible | Shows, disables, or hides built-in editor features. |
| `enableModeToggle` | `false` | Shows editor-level light/dark/system toggle. |
| `mode` | `"light"` | Initial editor mode. |
| `theme` | `"default"` | Built-in editor color theme. |
| `wrapperClassName` | - | Classes applied to the outer RichTextEditor wrapper, including preview tabs, mode toggle, preview panel, and editor shell. |
| `editorClassName` | - | Classes applied to the editor shell containing the menubar, editable content, drag handle, and word count footer. |

## RenderJSONProps

```ts
type RenderJSONProps = {
  content: JSONContent;
  immediatelyRender?: boolean;
  className?: string;
  mode?: RichTextEditorMode;
  enableTableOfContents?: boolean;
  tableOfContentsPosition?: "left" | "right";
};
```

| Property | Default | Description |
| --- | --- | --- |
| `content` | Required | Tiptap JSON content from `editor.getJSON()` or `getEditorContent().json`. |
| `immediatelyRender` | `false` | Controls immediate rendering for SSR-sensitive pages. |
| `className` | - | Classes applied to the root `.tr-editor` RenderJSON wrapper. |
| `mode` | `"system"` | Light, dark, or system mode for rendered JSON output. |
| `enableTableOfContents` | `false` | Shows a generated table of contents beside rendered JSON. |
| `tableOfContentsPosition` | `"right"` | Controls TOC placement. |

## Common Patterns

### Type-Safe Save

```tsx
import type { OnSaveArgs } from "tiptap-react-ui";

function handleSave(content: OnSaveArgs) {
  saveDocument(content);
}

<RichTextEditor editor={editor} onSave={handleSave} />;
```

### Loading State

```tsx
const { editor, isEditorLoading } = useTiptapEditor();

if (isEditorLoading) return <div>Loading editor...</div>;

return <RichTextEditor editor={editor} />;
```

### Extension State

```tsx
<RichTextEditor
  editor={editor}
  extensionState={{
    imageUpload: "hidden",
    bold: "disable",
  }}
/>
```

### Styling Layers

```tsx
const { editor } = useTiptapEditor({
  className: "min-h-[420px] overflow-y-auto p-6",
});

return (
  <RichTextEditor
    editor={editor}
    wrapperClassName="mx-auto max-w-5xl"
    editorClassName="rounded-xl border bg-background"
  />
);
```
