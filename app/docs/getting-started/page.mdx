# Tiptap React UI

Build beautiful and modern rich text editors for React with minimal setup.

`tiptap-react-ui` is a pre-built WYSIWYG editor powered by [Tiptap](https://tiptap.dev), [Tailwind CSS](https://tailwindcss.com), and [shadcn/ui](https://ui.shadcn.com), designed for developers who want a clean, customizable, and production-ready editing experience out of the box.

Whether you're building a blog platform, CMS, documentation site, admin dashboard, or note-taking application, `tiptap-react-ui` provides the essential tools you need to ship faster without building an editor from scratch.

---

## ✨ Features

### 🧩 Plug and Play

Get started quickly with pre-built editor components and minimal configuration.

### 📝 Modern Rich Text Editing

Includes essential formatting tools for a complete WYSIWYG editing experience.

### 🌙 Dark Mode Support

Seamlessly adapts to both light and dark themes.

### 💻 Syntax Highlighted Code Blocks

Write and display code beautifully with built-in syntax highlighting support.

### 🔢 Character & Word Count

Track content length in real time.

### 📄 Flexible Content Rendering

Preview and render both **JSON** and **HTML** content effortlessly.

### ⚡ Built for Modern React Apps

Designed with modern tooling using React, Tailwind CSS, and shadcn/ui.
