# Tiptap React UI

Build beautiful and modern rich text editors for React with minimal setup.

`tiptap-react-ui` is a pre-built editor UI powered by [Tiptap](https://tiptap.dev), designed for developers who want a clean, customizable, and production-ready editing experience out of the box.

Whether you're building a blog platform, CMS, documentation site or note-taking application, `tiptap-react-ui` provides the essential tools you need to ship faster without building an editor from scratch.

---

## Features

### 🚀 Plug and Play

Get started quickly with pre-built editor components and minimal configuration.

### ✨ Modern Rich Text Editing

Use `RichTextEditor` for a complete WYSIWYG editing interface with toolbar controls, bubble menus, preview tabs, save callbacks, word count, image upload, YouTube embeds, tables, and code blocks.

### 📝 Notion-like Editing

Use `NotionEditor` when you want a lighter Notion-style writing surface with slash commands, block controls, contextual menus, and a document-first editing feel.

### 🎨 Theme Support

`RichTextEditor`, `RenderJSON`, and `RenderHTML` support light, dark, and system modes. `RichTextEditor` also supports built-in color themes.

### 💻 Syntax Highlighted Code Blocks

Write and display code beautifully with built-in syntax highlighting support.

### 📊 Character & Word Count

Track content length in real time with the RichTextEditor footer.

### 🔄 Flexible Content Rendering

Preview, save, and render both JSON and HTML content. For persisted editor content, `RenderJSON` is recommended because it preserves the Tiptap document structure.

### 📚 Table Of Contents

Generate a live table of contents for JSON previews and rendered JSON output.
